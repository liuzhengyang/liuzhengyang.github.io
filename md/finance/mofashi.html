# 稳固期

价格均衡模式。好的股票会让纠正期的下跌维持在较小的服务端。之后，股票会继续休息并在之后结合着长期的 上涨趋继续上升。

# 波动收缩

在稳固期其价格波动不断收窄，同时伴随着交易量明显的收缩。 波动收缩规律（VCP）

转入上涨模式、交易量开始放大的时候买入。

# 深度纠正模式 - 失败的前兆

当股票快速下跌时，很可能暗示着公司或者行业出了某些问题，甚至可能心的熊市就要到来。 不能在此时因为股价从高点下跌50%甚至60%而认为股票价格便宜值得购买。

# 时间压缩

如果股票上涨速度过快，就会形成一个危险的时间压缩现象，而大多时候我们应该避免这种事情的发生。

# 关注需求的迹象

第二阶段是最佳的买入时机。 股票会因为人们将获得的回报变现而出现纠正（下跌）。这时，有些人会因为时机 选择的实物而错过卖出的最优时间。当股票再次上涨时，他们会抓住下次机会脱身离场。 卖出的速度放缓后，股票还有吸引力么？只要仔细观察价格和交易量的走势，我们就能对上述 问题给出自己粗略的答案。 要将视线集中起来，寻找机构投资者出现的痕迹。痕迹质疑就是价格的骤然升高。 价格拔地而起的情况一般发生在纠正期股价的地步或是纠正期后期。这类价格庆伟一般伴随着突然放大的交易量。 价格和交易量的突然升高通常暗示机构的买入行为。

# 合理的法则让你的视线更透彻

两次上涨一次下跌

下跌的越多，需要上涨回本的上涨比例成倍增加。

接受市场的判断 知道何时自己是错的。 避免巨大错误发生。 不要成为无意识的投资者。 因为投资者们决绝承认错误，他们会讲自己的错误合理化。 当交易朝着不利于自己的方向移动，并出现亏损时，他们会突然决定长期持有这些股票。 这时，他们就变成了利弗莫尔常说的"无意识的投资者"，一个只要有一点收益就收割，却把 损失仍在那里放任其放大的投资者。 利用止损规则限制了你的亏损后，你自己已经领先于 其他投资者了，因为大多数投资者都不敢做到这一点。 最好的交易以上那些能人事错误并用于止损，保护自己的资产， 等待下一次机会的人。

# 风险管理：如何处理并掌控风险

风险是可能损失的概率。 股票交易的模板上通过让潜在回报高于潜在风向来实现 持续盈利。但是，大多数投资者面临的问题是他们过于专注 于回报而忽视了风险。 自律。

# 养成习惯

# 何时应该斩仓止损

止损线应该与预期收益有关。

# 避开交易者的原罪

允许损失超过平均收益的行为我们称之为"交易者原罪"。 你必须让你的平均收益超过损失。但在现实情况中， 很多投资者设置不知道自己的平均收益是多少。 当设置止损线时，我的第一法则上让损失不能超过该 投资者历史成功投资中平均收益的一半。

同样建议大多数投资者，不论平均收益是多少，不要让 股票跌幅超过10%。

你能做的最好的事情就是让你的损失小于收益。

将亏损保持在较小额度上，并能成功避开投资者原罪。 永远不要让损失增长得大于你的平均收益。

# 向你的止损线致敬

在股市中，股价下跌的越多，其上涨的困难也就越大。

# 一定会带来灾难的习惯

用赚来的钱来填补亏损的投资上让你走向贫穷最快的方法。这叫探底成功。经济商通常 会劝其客户在股价下跌后继续买入加仓，以掩盖之前错误的投资建议。他们会告诉客户这样可以降低 其平均成本。但是股价如果继续下跌的话，你的损失会被放大到原来的两倍。 在股票交易中遭受损失没什么可羞耻的，但继续持有亏损的股票，放任损失放大，设置还在不断加仓 才是业余的、自我毁灭的行为。

对很多投资者来说，在价格回撤时买入股票上十分冒险的，尽管你会觉得与之前相比， 现在的价格更有诱惑力。但是，摊低成本行为会让你失败、平庸。

# 逐步建仓与摊低成本

专业与业余最关键的区别之一就是专家们在逐步创建仓位的时候，业余投资者正在摊低成本。

假设一个专家和业务选手都决定将5%的资本用于风险投资。专家可能会首选买2%，然后再买2%，最好再讲剩下的1%投资进去。 之后他可能会讲止损线设置在平均买入价以下10%的位置，总风险控制在其总资本的0.5%。 业余投资者会在同一价格一次性建仓完毕，如果市场走势与预期不同，他可能会讲仓位翻番，一次摊低平均成本。

永远不要相信初始价格，除非你的仓位已经让你盈利。

风险优先的方式看待每一笔交易。

# 永远不要让风险超过预期收益

止损线控制在止盈线一半以下。 平均成功率。 在50%的平均成功率下，你正确的次数和错误的次数一样多，所以要保持2:1的收益/风险比，你需要让亏损只有收益的一半。 然而，如果成功率是40%，为了保持相同的2:1的收益/风险比率，你的亏损必须保持在收益水平的1/3。
