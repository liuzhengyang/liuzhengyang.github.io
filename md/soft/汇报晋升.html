# 汇报晋升

工作、能力、思考

事情作为载体，体现能力。

# 问题挑战

自己能力范围hold住的项目。 如何解决一个有挑战的难题。

# 把自己当成产品

FAB

  • Feature 特性，内容
  • Advantage 优点，对照（友商、过去）
  • Benefit 价值视角

# 思考

  • 思考要前瞻
  • 胜任未来，未来1-2年的思考；对赛道、专业线的思考

# 逻辑框架

  • 履历概述定位

  • 履历，能力的提升成长

  • 概述，年度工作的大画面。关键指标、主题句。量化和非量化的。

  • 定位，价值与能力的边界。

  • 挑战和冲突：为评价提供了基准。设定基准，看到正向成绩。变化。

  • 困难与压力

  • 问题分析与解决

  • 原则/方法/努力/资源/机遇

  • 成果与收获：个人视角

  • 经验与传承：经验、沉淀的方法论，对公司的价值。组织视角。

挑战-方法-成果 三者要match

前瞻

  • 机会点、创新点

  • 挑战点、风险点

  • 总结过去是能力

  • 看到未来是眼光

# 回答问题环节

误区

  • 答非所问：放松+聆听，没听懂，可求证。“抱歉我没抓住您的要点”
  • 不懂装懂：坦诚相告、简单回应。“抱歉这块我还思考不够”
  • 表达混乱：

# 提前思考：他们会问什么

  • 听众关键问题清单
  • 问题-答案 PPT。问题问的好、重要。（不要回答太快，有思考时间，慢慢说）

# 回答技巧

  • 去掉防御心态

  • 接住问题：感谢提问、问题非常重要

  • 回答问题：这个问题我有三点想法/三个思路

  • 梳理一下

# 原型工具

  • 克服紧张：内容再简明清晰一些；多练习几次。
  • 贴纸、调整。
  • 动嘴说，而不是用脑子构建演讲稿。小黄鸭调试法。
  • storyboarding（做故事版）：语言先行，视觉助记。合适粒度，内容骨架（结构）。内容梳理，增加练习。便签。
