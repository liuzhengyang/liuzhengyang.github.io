# CSS

CSS(Cascading Style Sheet)是控制网页元素样式的技术。

# CSS语法

格式

选择器 {
    属性key: 属性值;
    属性key: 属性值;
    属性key: 属性值;
    属性key: 属性值;
}

# 选择器类型

  • 元素选择器 - 选择指定类型的元素，p选择 <p>
  • id选择器 - 选择特定id的元素，#my-id 选择 <p id="my-id"/>
  • 类选择器 - 选择特定类的元素, .my-class选择 <p class="my-class"/>
  • 属性选择器 - 选择拥有特定属性的元素, img[src] 选择 <img src="myimage.png"/>
  • 伪(Pseudo)类选择器 - 选择特定状态下的元素， a:hover 仅在鼠标悬停在链接上时选择<a>

# 盒子模型

一个个盒子，设置尺寸、颜色、位置等等。

  • padding: 内边距
  • border: 边框
  • margin: 外边距，围绕元素外部的空间
