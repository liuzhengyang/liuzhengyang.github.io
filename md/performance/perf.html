# perf

# 介绍

perf是linux系统中，用来性能分析、诊断、观测、profile的性能分析工具。

  • 为什么kernel在on-CPU上消耗这么多CPU，什么代码导致的？
  • 导致CPU level 2 cache miss的代码是什么
  • CPU在memory IO上很慢吗
  • 哪些代码在申请内存？
  • 谁在导致TCP重传？
  • 有哪些kernel function被调用、多频繁？

sudo yum install -y perf

# 使用

perf命令执行时，默认需要root权限。如果使用时，提示权限不足，则需要修改系统配置

echo 0 > /proc/sys/kernel/perf_event_paranoid
1

使用命令格式

# perf list

perf list能够展示所有的event type

常用指标说明

  • instructions:
  • cycles
  • cache-misses
  • dTLB-loads
  • dTLB-load-misses
  • dTLB-stores
  • dTLB-store-misses
  • L1-dcache-load-misses:
  • L1-dcache-loads
  • L1-dcache-stores
  • L1-icache-store-misses
  • LLC-load-misses
  • LLC-loads
  • LLC-store-misses
  • LLC-stores

# perf stat

下面命令会每秒统计pid进程的cache-missed,cycles等指标，如果需要其他指标（通过perf list或搜索），可以通过逗号分隔加到后面。
