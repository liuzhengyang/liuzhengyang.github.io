# 晋升

# 职级和晋升的逻辑

有了职级，才会有晋升的概念。

职级的目的是给员工能力划分等级，在面试、工作中能够衡量员工工作水平价值，给与对应的薪酬，因为每个直接会有对应的薪酬范围。

除了面试入职时产生一个初始职级，在工作期间员工还可以通过答辩完成职级的提升。

对于员工而言，职级提升带来薪酬提升，找工作时新的工作也一般会比当前职级要高一点或匹配(薪酬同理）。

对于公司而言，设定这样的职级和晋升机制，能够让员工更加有奋斗动力，通过这样的机制引导员工提升自己做出更高的成绩。

对于每个职级，公司一般都会定义一些标准，比如负责独立模块开发，负责一个业务项目，负责一个大的团队，等等都是不同职级的能力要求。

在晋升时，要注意参考这样的职级标准，通过答辩向评委证明自己已经达到了最近的要求。
