# RoadMap

# 技术

  • Java

    • 基础知识
      • 多线程、并发：
        • 锁: ReentrantLock, ReentrantReadWriteLock, AQS, synchronized, volatile
        • 线程池, ThreadPoolExecutor, FutureTask
        • 同步器, CountDownLatch, Semaphore, CyclicBarrier
        • 并发集合框架, ConcurrentHashMap。
      • 设计模式
        • strategy
        • chain of responsible
    • JVM
      • GC：G1、运行时内存分布、堆外内存、内存问题排查。
      • bytecode(asm, bytebuddy, javassist)、类加载器、JIT、
      • hotswap
    • 协程
      • VirtualThread
      • 阿里dragonwell协程

  • 中间件、框架

    • RPC: dubbo, grpc。netty。
    • MQ: kafka, rocketmq
    • Cache: redis, memcached, localcache。
      • redis集群、高可用
      • memcached集群高可用。
    • spring: bean生命周期、spring启动过程。
    • DB: mysql、innodb。建表、索引。锁问题。
    • config: zk, raft, nacos.

  • 可靠性

    • 分布式事务: seata, rocketmq
    • 熔断、限流、降级、超时。sentinel。
    • 监控报警预案演练

  • 系统设计

    • 缓存数据库一致性问题
    • 读写热点问题: 缓存、聚合、前后时间利用。
    • 分布式事务
    • 分布式系统设计：DDID。
    • 分布式锁
    • 分布式id生成
    • kv、排序问题
    • 扩展性设计
    • DDD
